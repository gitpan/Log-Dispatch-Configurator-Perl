This is Perl module B<Log::Dispatch::Configurator::Perl>.


=head1 INSTALLATION

Log::Dispatch::Configurator::Perl installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Log::Dispatch::Configurator::Perl

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install
